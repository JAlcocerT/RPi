# ESP32 Internal Temperature Sensor — Explained

How the ESP32 built-in temperature sensor works, which modules support it, and how to detect if it's unavailable.

---

## Which modules support it

| Module | Core | Has internal temp sensor? |
|---|---|---|
| ESP-WROOM-32 / ESP32-D0WDQ6 | Xtensa LX6 | ✅ Yes |
| ESP32-WROOM-32U | Xtensa LX6 | ✅ Yes |
| ESP32-S2 | Xtensa LX7 | ❌ No |
| ESP32-S3 | Xtensa LX7 | ❌ No |
| ESP32-C3 | RISC-V | ❌ No |
| ESP32-H2 | RISC-V | ❌ No |

The sensor is only present on the **original ESP32** (Xtensa LX6 dual-core). All newer variants dropped it.

---

## How it works

The sensor is accessed via a ROM function built into the ESP32 chip:

```cpp
#ifdef __cplusplus
extern "C" {
#endif
uint8_t temprature_sens_read();  // note: typo in the name is Espressif's own
#ifdef __cplusplus
}
#endif
```

It returns the temperature in **Fahrenheit** as a `uint8_t`. Convert to Celsius:

```cpp
float getInternalTempC() {
    return (temprature_sens_read() - 32) / 1.8f;
}
```

---

## What it actually measures

- **Chip temperature** — not room temperature
- Reflects the heat generated by the ESP32 itself
- Expect it to read **10–20°C higher** than ambient when under load
- Accuracy: ±5°C typical — useful for overheat detection, not precise measurement
- Typical idle reading: ~50–55°C

---

## Detecting if the sensor is unsupported

If the sensor is not present on the chip, `temprature_sens_read()` returns `128`, which converts to exactly **53.33°C**. A simple sanity check:

```cpp
float chipTemp = getInternalTempC();
if (chipTemp > 80.0 || chipTemp < 0.0) {
    Serial.println("Warning: internal sensor may not be supported on this chip.");
}
```

A real idle reading should be between 30°C and 75°C. Outside that range, the sensor is likely absent or faulty.

---

## Limitations

| Limitation | Detail |
|---|---|
| Not calibrated | ±5°C error, reads relative not absolute |
| Chip temp only | Rises with CPU load and WiFi activity |
| Original ESP32 only | Not available on S2, S3, C3, H2 |
| ROM function | No official Espressif API — may break on future IDF versions |

## MQTT topic

```
esp32/temperature/internal   →   52.44  (°C, 2 decimal places)
```

Subscribe to verify:
```sh
mosquitto_sub -h 192.168.1.2 -t "esp32/temperature/#" -v
```

---

## Combine with DHT11 for context

The internal sensor is most useful alongside an external sensor — the difference between the two tells you how much the board is self-heating:

```
self_heating = chip_temp - dht11_ambient_temp
```

If `self_heating` is consistently above 20°C, the board may need better airflow or a heatsink.
